## Installation
  
1. - Clone the repo
 ```sh
  git clone https://github.com/ggab13/google-homepage.git
   ```

2. - Navigate to the project directory
```sh
cd google-homepage
 ```
3. - Open the index.html file in your web browser to view the Google homepage clone.
